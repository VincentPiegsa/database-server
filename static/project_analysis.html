Die Daten wurden im JSON-Format auf der SD-Karte gespeichert. Zur weiteren Analyse habe ich ein Python-Programm geschrieben, das die Messungsdaten Zeile für Zeile in eine SQLite3-Datenbank schreibt. Die dadurch erzeugte portable .db Datei konnte dann mit einem SQLite-Browser geöffnet werden und mit SQL-Befehlen bearbeitet und analysiert werden. Zur Visualisierung habe ich ebenfalls Python genutzt, allerdings mit dem Modul matplotlib, das Grafiken aus Daten erzeugt. Die Programme und Grafiken befinden sich in meinem GitHub Repository.
