Die Daten, die auf dieser Datenbank zu finden sind, wurden mit einer selbst gebauten Wetterstation, basierend auf einem Arduino, gemessen. Dazu wurde der DHT11, ein Temperatur- und Luftfeuchtigkeitssensor genutzt, um das Klima aufzuzeichnen. Die Messungsdaten wurden auf eine SD-Karte geschrieben und mit Datum und Uhrzeit datiert. Zur parallelen grafischen Darstellung der Messung wurde ein 16x2 LCD angeschlossen.

Näheres zu den Programmen und Schaltplänen befindet sich ebenfalls in meinem GitHub Repository und im Protokoll.
