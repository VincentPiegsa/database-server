Um die Datenbank über eine Weboberfläche nutzen zu können, wurde ein Programm benötigt, das die Eingabe aus dem Textfeld in die Datenbank einsetzt und das Resultat zurückliefert. Falls beim Ausführen der Query ein Fehler auftritt, wird die Fehlermeldung auf der Seite angezeigt. Dafür gab es mehrere Möglichkeiten:

django: Ein Python Framework node.js. Eine JavaScript-Laufzeitumgebung Go: Eine Programmiersprache, die hauptsächlich für Server genutzt wird

Letztendlich habe ich mich für node.js entschieden, da es verständlicher als django und vielseitiger als Go ist.
